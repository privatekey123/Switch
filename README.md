# Switch Desktop (macOS + Windows)

Desktop-App für einen Preview/Program-Workflow:

- Auswahl **aller verfügbaren Fenster & Bildschirme**
- **Program** (Live) und **Preview** (Next)
- **CUT** und **FADE**
- Separates **Program-Fenster** für externen Monitor
- **Favoriten** speichern/laden

## Voraussetzungen

- Node.js 18+
- macOS oder Windows

## Start

```bash
npm install
npm start
```

## Build-Hinweis

Für echte Installer (`.dmg` / `.exe`) kann als nächster Schritt `electron-builder` ergänzt werden.
